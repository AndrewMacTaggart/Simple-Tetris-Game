Mathematical Formulas Reference

Algebra

Quadratic Formula

For ax² + bx + c = 0:

x = (-b ± √(b² - 4ac)) / 2a

Distance Formula

Distance between two points:

d = √[(x₂ - x₁)² + (y₂ - y₁)²]

Slope Formula

Slope of a line:

m = (y₂ - y₁) / (x₂ - x₁)

Pythagorean Theorem

For a right triangle:

a² + b² = c²
